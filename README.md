# ValueSelectorLib

Numeric TextView with add & minus btn for Android.

##Usage

###Gradle

```gradle
allprojects {
 repositories {   
    maven { url "https://jitpack.io" }
 }
}
```
and
```gradle
dependencies {
    compile 'com.github.mehdi-salehi:valueselectorlib:1.0'
}
```
